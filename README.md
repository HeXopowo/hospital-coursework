# Hospital Management System
Система управления больницей – десктопное приложение для автоматизации работы регистратуры, врачей и администраторов. 
Позволяет вести учёт пациентов, врачей, запись на приёмы, формировать медицинские карты, рецепты, а также архивировать данные.

## Назначение
- Автоматизация записи пациентов на приём к врачам.
- Ведение базы данных пациентов (ФИО, контакты, СНИЛС, полис ОМС, участок).
- Ведение базы данных врачей (специализация, кабинет, график работы, email).
- Управление приёмами (создание, редактирование, отмена, поиск).
- Ведение медицинских записей (диагнозы, рекомендации, анамнез).
- Выписка рецептов.
- Постановка пациентов на учёт к врачу.
- Архивация удалённых записей с возможностью восстановления.
- Разграничение ролей: администратор, врач, пациент.

## Стек технологий
Язык программирования: Java 21
GUI: JavaFX 21, FXML, CSS
Система сборки: Apache Maven 3.8+
База данных: PostgreSQL 14+
JDBC: PostgreSQL JDBC Driver 42.6.0
Тестирование: JUnit 5, Mockito
Контроль версий: Git

## Установка и запуск
### Требования
- JDK 21 (или выше)
- PostgreSQL 14+ с созданной базой данных
- Maven 3.8+ (или встроенный в IDE)
  
### Шаги
1. Клонируйте репозиторий
   git clone https://github.com/username/hospital-app.git
   cd hospital-app
2. Отредактируйте файл src/main/java/hospital/DatabaseConnection.java, укажите свои параметры:
private static final String URL = "jdbc:postgresql://localhost:5432/ваша_бд";
private static final String USER = "ваш_пользователь";
private static final String PASSWORD = "ваш_пароль"; 
3. Восставновите базу данных с помощью приложенного backup-файла
4. Запустите с помощью maven mvn clean javafx:run

### Основные возможности
Администратор: управление врачами, пациентами, учётными записями, архивом, просмотр всех приёмов.
Врач: просмотр своих пациентов, запись на приёмы, ведение медкарт, выписка рецептов, постановка на учёт.
Пациент: просмотр своих приёмов, медкарт, рецептов, поиск врачей.

### Пример рабочего процесса
Администратор добавляет врача и пациента.
Пациент (или администратор) записывается на приём к врачу.
Врач проводит приём, заполняет медицинскую запись, при необходимости выписывает рецепт или ставит на учёт.
Пациент может просмотреть свою медкарту и рецепты.
При удалении врача/пациента данные перемещаются в архив; администратор может восстановить их.
